# Agent Move: Python

## Step 1
Code an ``||player:on chat||`` command and name it **“tp”**.

```python
def on_chat():
    pass
player.on_chat("tp", on_chat)
```

## Step 2

Inside the **tp**  ``||player:on chat||`` command, code the agent to ``||agent:teleport to player||`` .

```python
def on_chat():
    agent.teleport_to_player()
player.on_chat("tp", on_chat)
```
